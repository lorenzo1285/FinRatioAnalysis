
# FinRatioAnalisys

Finacial Ratio Analysis Project: 

-Return Analysis

-Margin Analysis

-Leverage Analysis

-Liquidity Analysis

## Documentation

[Documentation](https://corporatefinanceinstitute.com/assets/CFI-Financial-Ratios-Cheat-Sheet-eBook.pdf)

Quick Start.

Packages:

pip install yahoo-fin

pip install pandas

pip install FinRatioAnalysis

Import:

from FinRatioAnalysis import FinRatioAnalysis

Create an Objet:

ra = FinRatioAnalysis('ticker')

Available methods:

ra.Return_Ratios()

ra.Margin_Ratios()

ra.Leverage_Ratios()

ra.Liquidity_Ratios()
